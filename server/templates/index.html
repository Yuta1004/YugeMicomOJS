ジャッジステータス解説

YMOJSには8つのジャッジステータスが存在します

ジャッジステータス
 # 解説 得点
AC
 Accepted 正答です。提出コードが要求通りに動作することが確認されました ◎
WA
 Wrong Answer 誤答です。提出コードの出力が正しくありません。
TLE
 Time Limit Exceeded 制限時間内に提出コードの実行が終了しませんでした。
RE
 Runtime Error 実行エラーです。提出コードの実行が正常に終了しませんでした。
CE
 Compilation Error コンパイルエラーです。スクリプト言語の場合は文法エラーです。
IE
 Internal Error サーバエラーです。システム運営者にリジャッジを申請してください。
WJ
 Waiting for Judge ジャッジ中です。
未提出
 未提出 未提出です。
